# УчётЛёгко

Простая онлайн-бухгалтерия для малого бизнеса.

## Structure

| Directory | Description |
|-----------|-------------|
| `app/` | Next.js 14 application (TypeScript, Prisma, PostgreSQL) |
| `landing/` | Static landing page |
| `docs/` | Documentation |
| `.github/workflows/` | CI/CD pipelines |

## Getting Started

See [docs/dev-setup.md](docs/dev-setup.md) for full setup instructions.

```bash
cd app && npm install && npm run dev
```

## Stack

- **Next.js 14** — fullstack React framework
- **TypeScript** — type safety
- **PostgreSQL + Prisma** — database
- **Vercel** — deployment
- **GitHub Actions** — CI/CD
